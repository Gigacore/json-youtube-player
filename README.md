json-youtube-player
===================

A simple JSON driven youtube player that can play videos via channel or custom data.


Setup
===================

1. Include jQuery.
2. Include json-player.js
3. Use the sample HTML snippet:

<div id="left_column">
	<div id="player_wrap"></div>
		<div id="meta">
			<div id="title">
					<h2></h2>
			</div>
			<div id="desc"></div>
		</div>
	</div>
	<aside>
		<ul id="videos"></ul>
	</aside>
</div>
